applylut

基于查找表的二值图像邻域运算

注：applylut 函数已不推荐使用，建议改用 bwlookup 函数替代。

简介

A = bwlookup(BW,lut)

用法

A = applylut (BW, lut)该函数借助查找表 lut 对二值图像 BW 执行 2×2 或 3×3 尺度的邻域运算。此查找表包含了所有可能的 2×2 或 3×3 邻域对应的输出值。

参数说明

输入参数

BW — 输入图像
二维二值图像

输入图像需指定为二维二值图像。若为数值型输入，所有非零像素均被视为 1（真）。

lut — 输出像素值查找表
16 元向量 | 512 元向量

输出像素值查找表需指定为 16 元或 512 元向量（格式与 makelut 函数的返回值一致）。

输出参数

A — 输出图像
二值图像 | 灰度图像

输出图像以灰度图像或二值图像形式返回，其像素值分布由查找表 lut 的内容决定；输出图像 A 与输入图像 BW尺寸完全一致。

若 lut 的所有元素均为 0 或 1，则 A 的数据类型为逻辑型；

若 lut 的所有元素均为 0 至 255 之间的整数，则 A 的数据类型为 8 位无符号整型；

其余所有情况，A 的数据类型均为双精度浮点型。
