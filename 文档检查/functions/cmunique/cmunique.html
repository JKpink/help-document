cmunique

消除颜色图中的重复颜色；将灰度或真彩色图像转换为索引图像

简介

[Y, newmap] = cmunique(X, map)
[Y, newmap] = cmunique(RGB)
[Y, newmap] = cmunique(I)

用法

[Y, newmap] = cmunique(X, map) 从颜色图 map 中删除重复的行以生成新颜色图 newmap。该函数还会调整强度图像 X 中的索引以保持索引与颜色图之间的对应，并在 Y 中返回结果。输出图像 Y 及其关联的颜色图 newmap 在视觉上与输入 X 和 map 等效，但使用了可能的最小颜色图。
[Y, newmap] = cmunique(RGB) 将真彩色图像 RGB 转换为索引图像 Y 及其关联的颜色图 newmap。返回的颜色图是图像的可能的最小颜色图，包含 RGB 中每种唯一颜色对应的一项。
[Y, newmap] = cmunique(I) 将灰度图像 I 转换为索引图像 Y 及其关联的颜色图 newmap。返回的颜色图是图像的可能的最小颜色图，包含 I 中每种唯一灰度级别对应的一项。

参数说明

输入参数

X — 具有重复颜色的索引图像
m×n 整数矩阵

具有重复颜色的索引图像，指定为 m×n 整数矩阵。

数据类型： double | uint8 | uint16

map — 具有重复颜色的颜色图
c1×3 矩阵

与索引图像 X 相关联的具有重复颜色的颜色图，指定为由范围 [0, 1] 内的值组成的 c1×3 矩阵。map 的每行都是一个三元素 RGB，指定颜色图的单种颜色的红、绿和蓝分量。

数据类型： double

RGB — RGB 图像
非负数组成的 m×n×3 数组

RGB 图像，指定为由非负数组成的 m×n×3 数组。

数据类型： double | uint8 | uint16

I — 灰度图像
m×n 数值矩阵

灰度图像，指定为 m×n 数值矩阵。

数据类型： double | uint8 | uint16

输出参数

Y — 具有唯一颜色的索引图像
m×n 整数矩阵

具有唯一颜色的索引图像，以 m×n 整数矩阵形式返回。如果 newmap 的长度小于或等于 256，则输出图像可以属于 uint8 类。否则，输出图像属于 double 类。

数据类型： double | uint8

newmap — 具有唯一颜色的颜色图
c2×3 矩阵

与输出索引图像 Y 相关联的具有唯一颜色的颜色图，返回为由范围 [0, 1] 内的值组成的 c2×3 矩阵。newmap 的每行都是一个三元素 RGB，指定颜色图的单种颜色的红、绿和蓝分量。

数据类型： double
