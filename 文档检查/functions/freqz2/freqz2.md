## freqz2
创建预定义的3维滤波器
## 简介
[ `[H, fx, fy] = freqz2(h)`](#function1)  
[ `[H, fx, fy] = freqz2(h, [nx, ny])`](#function2)  
[ `[H, fx, fy] = freqz2(h, [nx, ny], [dx, dy])`](#function3)  
[ `freqz2( __ )`](#function5)  
## 用法
<a id="function1"></a>
[[H](#Q1), [fx](#Q2), [fy](#Q3)] = freqz2([h](#Q4))返回二维 FIR 滤波器 `h` 的 64×64 频率响应 `H` （ `h` 需为相关核形式）；同时返回归一化频率向量 `fx` 、 `fy` 。  
<a id="function2"></a>
[[H](#Q1), [fx](#Q2), [fy](#Q3)] = freqz2([h](#Q4), [[nx, ny](#Q5)])返回二维 FIR 滤波器 `h` 的ny×nx尺寸频率响应 `H` ；也可将  `nx` 、 `ny` 作为两个独立参数传入  
<a id="function3"></a>
[[H](#Q1), [fx](#Q2), [fy](#Q3)] = freqz2([h](#Q4), [[nx, ny](#Q5)], [[dx, dy](#Q6)])用[`dx`,`dy`]覆盖h中的采样间隔；若传入标量，则 `x`、`y` 维度使用相同的采样间隔。  
<a id="function4"></a>
[H](#Q1) = freqz2([h](#Q4), [fx](#Q2), [fy](#Q3)) 	在指定的频率值 `fx` 、 `fy` 处，返回二维 FIR 滤波器 `h` 的频率响应 `H` 。   
<a id="function5      "></a> 
freqz2( __ )不指定输出参数时，会绘制二维幅度频率响应的网格图。

## 参数说明
### 输入参数
**<a id="Q1"></a> h — 二维 FIR 滤波器**  
数值矩阵  

以相关核形式存在的二维 FIR 滤波器，需指定为数值矩阵。  

**<a id="Q2"></a> nx,ny — 频率响应的点数**  
[64 64] (默认) | 正整数 | 2 元素正整数向量  

指定频率响应的点数，若为标量，则 `nx` 和 `ny` 使用同一数值；点数越多，频率响应的频域分辨率越高，但计算成本也相应增加。  

**<a id="Q3"></a> dx,dy — 采样间隔**  
0.5 (默认) | 数值标量 | 2 元素数值向量  

指定 x、y 维度的采样间隔（若为标量，则二维使用相同间隔）。  
### 输出参数
**<a id="P1"></a> H — 频率响应**  
数值数组  

返回二维 FIR 滤波器的频率响应，以数值数组形式呈现。  

**<a id="P2"></a> fx — 频率向量**  
向量  

返回 x 维度的归一化频率向量。  

**数据类型：** `double`  

**<a id="P3"></a> fy — 频率向量**  
向量  

返回 y 维度的归一化频率向量。  

**数据类型：** `double`
