geometricTransform2d

二维几何变换对象

简介

tform = geometricTransform2d(inverseFcn)
tform = geometricTransform2d(inverseFcn, forwardFcn)

用法

tform = geometricTransform2d(inverseFcn) 创建一个 geometricTransform2d 对象并设置逆映射 inverseFcn 属性。
tform = geometricTransform2d(inverseFcn, forwardFcn) 添加了正向映射 forwardFcn 属性。

参数说明

输入参数

inverseFcn — 逆映射函数
函数句柄

逆映射函数，指定为函数句柄。该函数的输入与输出必须遵循统一的数据结构：以 n×2 矩阵表示 n 个二维点的坐标集合，其中第一列为 x 坐标，第二列为 y 坐标。
示例： ifcn = @(xy) [xy(:,1).^2, sqrt(xy(:,2))];

inverseFcn — 正映射函数
函数句柄

正映射函数，指定为函数句柄。该函数的输入与输出必须遵循统一的数据结构：以 n×2 矩阵表示 n 个二维点的坐标集合，其中第一列为 x 坐标，第二列为 y 坐标。 示例： ffcn = @(xy) [sqrt(xy(:,1)),(xy(:,2).^2)];

输出参数

tform — 二维几何变换对象
geometricTransform2d 对象

二维几何变换对象，返回 geometricTransform2d 对象。geometricTransform2d 对象使用逐点映射函数定义自定义的二维几何转换。
