## graydiffweight
根据灰度强度差异计算图像像素的权重

## 语法
[`W = graydiffweight(I, refGrayVal)`](#function1)  
[`W = graydiffweight(I, mask)`](#function2)  
[`W = graydiffweight(I, C, R)`](#function3)  
[`W = graydiffweight(I, C, R, P)`](#function4)  
[`W = graydiffweight(__, Name, Value)`](#function5)

## 说明
<a id="function1"></a>
[W](#Q1) = graydiffweight([I](#Q2), [refGrayVal](#Q3)) 计算灰度图像 `I` 中每个像素的像素权重，该权重是像素强度与由标量 `refGrayVal` 指定的参考灰度强度之间的差的绝对值，选择一个具有代表性的参考灰度强度值，以表示您要分割的目标。权重返回在数组 `W` 中，其大小与输入图像 `I` 相同。像素的权重与其灰度强度差的绝对值成反比。如果差异较小(强度值接近 `refGrayVal`)，则权重值较大，如果差异较大(强度值与 `refGrayVal` 相差甚远)，则权重值较小。  
<a id="function2"></a>
[W](#Q1) = graydiffweight([I](#Q2), [mask](#Q4)) 计算像素权重，其中参考灰度值为图像I中所有在掩码 `mask` 里标记为逻辑真（true）的像素的灰度值均值。与前一种语法中使用单一参考灰度值相比，使用多个像素的均值作为参考灰度值往往更有效。  
<a id="function3"></a>
[W](#Q1) = graydiffweight([I](#Q2), [C](#Q5), [R](#Q6))  基于二维灰度图像 `I` 中选定像素的灰度均值作为参考值，计算像素权重。选定像素的列索引和行索引分别为 `C` 和 `R` 。  
<a id="function4"></a>
[W](#Q1) = graydiffweight([I](#Q2), [C](#Q5), [R](#Q6), [P](#Q7)) 基于三维灰度体图像I中选定体素的灰度均值作为参考值，计算体素权重。选定体素的列索引、行索引和平面索引分别为 `C` 、`R` 和 `P` 。  
<a id="function5"></a>
[W](#Q1) = graydiffweight(__, [Name, Value](#Q8))  使用额外的名称值参数控制权重计算的相关特性。

## 参数说明
### 输入参数
**<a id="Q2"></a> I — 灰度图像**  
二维或三维数值数组

灰度图像，指定为二维或三维数组矩阵。

**数据类型：** `single` | `double` | `int8` | `int16` | `uint8` | `uint16` | `int32` | `uint32`

**<a id="Q3"></a> refGrayVal — 参考灰度强度值**  
标量

参考灰度强度值，指定为标量。

**数据类型：** `double`

**<a id="Q4"></a> mask — 参考灰度强度掩码**
逻辑数组

参考灰度强度掩码，指定为与 `I` 大小相同的逻辑数组。

**数据类型：**`logical`

**<a id="Q5"></a> C — 参考像素（或体素）的列索引**
正整数向量

参考像素的列索引，指定为正整数向量。

**数据类型：** `double`

**<a id="Q6"></a> R — 参考像素（或体素）的行索引**
正整数向量

参考像素的行索引，指定为正整数向量。

**数据类型：** `double`

**<a id="Q7"></a> P — 参考体素的平面索引**
正整数向量

参考体素的平面索引，指定为正整数向量。

**数据类型：** `double`

**<a id="Q8"></a> Name,Value — 命名值参数**
可选参数对需按 Name1, Value1,…, NameN, ValueN 的形式指定，其中 Name 为参数名称，Value 为对应的参数值。名称 - 值对参数必须出现在其他参数之后，但参数对的排列顺序不影响执行结果。

**RolloffFactor —— 输出权重衰减因子**
默认值为0.5 | 正数

输出权重衰减因子，指定为正数。该参数控制权重值随像素强度值与参考灰度值的绝对差值变化的衰减速率。将像素强度值与参考灰度值的绝对差值与权重值的关系绘制成二维曲线时，区域边缘处的像素强度值可能会平缓变化，形成平缓的斜率；而在分割后的图像中，你可能希望边缘的界定更清晰。通过设置该衰减因子，可控制 “像素强度值开始变化位置处” 的权重值曲线斜率：若指定较高的衰减因子值，平滑区域边缘附近的输出权重值会急剧下降；若指定较低的值，平滑区域边缘附近的输出权重值衰减会更平缓。该参数的建议取值范围为 [0.5  4]。

**数据类型：**`double`

**GrayDifferenceCutoff — 灰度强度绝对差值阈值**
默认为Inf | 非负数

灰度强度绝对差值阈值：指定为一个非负数值。该参数的默认值为Inf（无穷大），即表示不设置硬截断阈值。

当你应用该阈值时，会强抑制那些像素强度值远大于或远小于参考值的像素权重。`graydiffweight` 会将这些像素的权重值赋值为最小权重值。


**数据类型：**`double`

### 输出参数
**<a id="Q1"></a> W — 权重数组**  
数值数组

权重数组，指定为与输入图像 `I` 大小相同的数值数组。

**数据类型：** `double`
