## imnlmfilt
图像的非局部均值滤波

## 简介
[ `J = imnlmfilt(I)`](#function1)

## 用法
<a id="function1"></a>
[J](#Q2) = imnlmfilt([I](#Q1)) 对灰度或彩色图像 `I` 应用非局部均值滤波器，并在 `J` 中返回结果图像。

## 参数说明
### 输入参数
**<a id="Q1"></a> I — 要过滤的图像**  
2D 灰度图像 | 2D 彩色图像

要过滤的图像，指定为大小为 m×n 的 2D 灰度图像或大小为 m×n×3 的 2D 彩色图像。`I` 的大小必须大于或等于 21×21。

**数据类型：** `uint8`

### 输出参数
**<a id="Q2"></a> J — 非局部均值滤波图像**  
2D 灰度图像 | 2D 彩色图像

非局部均值滤波图像，返回为与输入图像 `I` 具有相同大小和数据类型的 2D 灰度图像或 2D 彩色图像。

## 算法
- 要平滑 RGB 图像中的感知接近颜色，请在应用非局部均值过滤器之前使用 rgb2lab 将图像转换为 CIE L\*a\*b\* 色彩空间。要查看结果，请首先使用 <a href="../lab2rgb/lab2rgb.html">lab2rgb</a> 将过滤后的 L\*a\*b\* 图像转换为 RGB 色彩空间。
