imnoise

向图像添加噪声

简介

J = imnoise(I, "gaussian")
J = imnoise(I, "gaussian", m)
J = imnoise(I, "gaussian", m, var_gauss)
J = imnoise(I, "localvar", var_local)
J = imnoise(I, "localvar", intensity_map, var_local)
J = imnoise(I, "poisson")
J = imnoise(I, "salt & pepper")
J = imnoise(I, "salt & pepper", d)
J = imnoise(I, "speckle")
J = imnoise(I, "speckle", var_speckle)

用法

J = imnoise(I, "gaussian") 对灰度图像 I 叠加均值为零、方差为 0.01 的高斯白噪声。
J = imnoise(I, "gaussian", m) 在图像 I 上叠加均值为 m、方差为 0.01 的高斯白噪声。
J = imnoise(I, "gaussian", m, var_gauss) 对图像 I 叠加均值为 m、方差为 var_gauss 的高斯白噪声。
J = imnoise(I, "localvar", var_local) 对图像 I 叠加局部方差为 var_local 的零均值高斯白噪声。
J = imnoise(I, "localvar", intensity_map, var_local) 叠加零均值高斯白噪声，其局部方差 var_local 通过强度映射表与图像 I 的像素强度相关联，该映射由向量 intensity_map 定义。
J = imnoise(I, "poisson") 从数据中生成泊松噪声，而不是向数据中添加人为噪声。
J = imnoise(I, "salt & pepper") 以默认密度 0.05 向图像 I 中叠加椒盐噪声，预计约 5% 的像素将受影响。
J = imnoise(I, "salt & pepper", d) 按指定噪声密度 d 叠加椒盐噪声，预计受影响像素数约为 d*numel(I)。
J = imnoise(I, "speckle") 使用方程 J = I+n*I 添加乘性噪声，其中 n 是均值为 0、方差为 0.05 的均匀分布随机噪声。
J = imnoise(I, "speckle", "var_speckle") 添加方差为 var_speckle 的乘性噪声。

参数说明

输入参数

I — 灰度图像
数值数组

灰度图像，指定为任意维度的数值数组。 对于 double 或 single 类型的输入图像，imnoise 函数要求其像素值处于归一化区间 [0, 1]。若输入数据超出该范围，函数会在噪声添加阶段自动执行边界裁剪。为确保处理过程的数值稳定性与预期效果，建议在处理流程中预先使用 rescale 函数完成数值范围的正规化。

数据类型： single | double | int16 | uint8 | uint16

m — 高斯噪声的均值
0（默认）| 数值标量

高斯噪声的均值，指定为数值标量。

var_gauss — 高斯噪声的方差
0.01（默认）| 数值标量

高斯噪声的方差，指定为数值标量。

var_local — 高斯噪声的局部方差
数值数组 | 数值向量

高斯噪声的局部方差，指定为数值数组或数值向量。如果指定 intensity_map 参量，则 var_local 必须为长度相同的向量。否则，var_local 必须为与输入图像 I 大小相同的数组。

intensity_map — 图像强度到噪声方差的映射
数值向量

图像强度到噪声方差的映射，指定为数值向量。imnoise 函数将图像强度值归一化到范围 [0, 1]。

d — 噪声密度
0.05（默认）| 数值标量

椒盐噪声的噪声密度，指定为数值标量。噪声应用于大约 d*numel(I) 个像素。

var_speckle — 乘性噪声的方差
0.05（默认）| 数值标量

乘性噪声的方差，指定为数值标量。

输出参数

J — 含噪图像
数值矩阵

含噪图像，以与输入图像 I 具有相同数据类型的数值矩阵形式返回。对于数据类型为 double 或 single 的图像，imnoise 函数在添加噪声后将输出像素值裁剪到范围 [0, 1]。
