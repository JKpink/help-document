imsharpen

使用非锐化模板锐化图像

简介

B = imsharpen(A)
B = imsharpen(A, Name, Value)

用法

B = imsharpen(A) 使用钝化蒙版方法锐化灰度或真彩色(RGB)图像 A 。
B = imsharpen(A, Name, Value) 使用名称-值参数来控制钝化蒙版的各个方面。

参数说明

输入参数

A — 待锐化图像
灰度图像 | RGB 图像

要锐化的图像，指定为灰度或 RGB 图像。

数据类型: uint8

名称-值参数

指定可选的参数对，格式为 Name1, Value1,...,NameN, ValueN，其中 Name 是参数名称，Value 是对应的值。名称 - 值参数必须出现在其他参数之后，但参数对的顺序无关紧要。

Radius — 高斯低通滤波器的标准偏差
1（默认值）| 正数

高斯低通滤波器的标准偏差，指定为正数。此参数控制边缘像素周围受锐化影响的区域大小。较大的值会使边缘周围更宽的区域变锐，而较小的值会使边缘周围更窄的区域变锐。

数据类型： uint8

Amount — 锐化效果强度
0.8（默认值）| 数字

锐化效果的强度，以数字形式指定。该值越大，锐化像素的对比度增加幅度越大。此参数的典型值在 [0, 2] 范围内，但允许使用大于 2 的值。此参数的值过大可能会在输出图像中产生不良效果。

数据类型： uint8

Threshold — 被视为边缘像素的最小对比度
0（默认值）| 范围 [0, 1] 内的数字

被视为边缘像素的最小对比度，指定为范围 [0, 1] 内的数字。较大的值（更接近 1）仅允许在高对比度区域（如强边缘）进行锐化，而不会影响低对比度区域。较小的值（更接近 0）还允许在图像相对平滑的区域进行锐化。此参数有助于避免在输出图像中锐化噪声。

数据类型： uint8

输出参数

B — 锐化的图像
数字数组

锐化图像，与输入图像 A 的大小和数据类型相同。
