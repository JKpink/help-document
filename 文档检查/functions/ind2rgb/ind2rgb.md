## ind2rgb
将索引图像转换为 RGB 图像

## 简介
[`RGB = ind2rgb(X, map)`](#function1)  

## 用法
<a id="function1"></a>
[RGB](#P1) = ind2rgb([X](#Q1), [map](#Q2)) 函数将索引图像 `X` 与其关联的颜色映射表 `map` 进行格式转换，输出等价的真彩色图像。

## 参数说明
### 输入参数
**<a id="Q1"></a> X — 索引图像**  
m×n 整数矩阵

索引图像，指定为 m×n 整数矩阵。

- 如果将 X 的指定为整数数据类型的数组，则值 0 对应于颜色映射表 [map](#Q2) 中的第一种颜色。对于包含 c 种颜色的颜色映射表，图像 X 的值会被裁剪到范围 [0, c-1] 内。
- 如果将 X 的数据类型指定为 `single` 或 `double` 的数组，则值 1 对应于颜色映射表中的第一种颜色。对于包含 c 种颜色的颜色映射表，图像 X 的值会被裁剪到范围 [1, c] 内。

**数据类型：** `single` | `double` | `uint8` | `uint16`

**<a id="Q2"></a> map — 颜色映射表**  
c×3 矩阵

与索引图像 [X](#Q1) 关联的颜色映射表，指定为取值归一化至 [0, 1] 区间的 c×3 矩阵。map 每行对应一组 RGB 三元色值，用于定义该颜色映射表中单个颜色的红、绿、蓝通道分量。 

**数据类型：** `double` 

### 输出参数
**<a id="P1"></a> RGB — RGB 图像**  
m×n×3 数值数组

真彩色图像以取值归一化至 [0, 1] 区间的 m×n×3 数值数组形式返回；RGB 的第三维度为颜色通道维度，为每个像素依次定义红、绿、蓝三通道亮度值。

**数据类型：** `double`
