## randomAffine2d
创建随机二维仿射变换

## 简介
[`tform = randomAffine2d`](#function1)  
[`tform = randomAffine2d(Name, Value)`](#function2)

## 用法 
<a id="function1"></a> [tform](#P1) = randomAffine2d 创建一个执行恒等变换的 <a href="../affinetform2d/affinetform2d.html">affinetform2d</a>  </a> 对象。  
<a id="function2"></a> [tform](#P1) = randomAffine2d([Name, Value](#Q1)) 使用名称-值参量指定二维仿射变换的类型。

## 参数说明
### <a id="Q1"></a> 名称-值参数
将可选的参量对组指定为 Name1, Value1,..., NameN, ValueN，其中 `Name` 是参量名称，`Value` 是对应的值。名称-值参量必须出现在其他参量后，但对各个参量对组的顺序没有要求。

**示例：** tform = randomAffine2d("XReflection", true)  
  
**XReflection — 随机水平翻转**  
`false`(默认) | `true`  
  
随机水平翻转，指定为 `false` 或 `true`。当 `XReflection` 设为 `true` 时，变换对象 [tform](#P1) 会以 50% 的概率对图像进行水平翻转。默认情况下，变换不会对图像进行水平方向翻转。  
  
**YReflection — 随机垂直翻转**  
`false`(默认值) | `true`  
  
随机垂直翻转，取值为 `false` 或 `true` 。当 `YReflection` 设为 `true` 时，变换对象 [tform](#P1) 会以 50% 的概率对图像进行垂直翻转。默认情况下，变换不会对图像进行垂直方向翻转。    

**Rotation — 旋转范围**  
[0 0]（默认）| 2 元素数值向量 | 函数句柄  

应用于输入图像的旋转角度范围，单位为度，指定为以下形式之一：  
  
- 2 元素数值向量：第二个元素必须大于或等于第一个元素。旋转角度将从指定区间内的连续均匀分布中随机选取。  
- 函数句柄：该函数必须无输入参数，并返回一个数值标量作为旋转角度。可通过函数句柄实现从不连续区间或非均匀概率分布中选取旋转角度。  

默认情况下，变换 [tform](#P1) 不执行图像旋转操作。

**示例：** [-45 45]

**Scale — 等比缩放范围**  
[1 1]（默认）| 2 元素数值向量 | 函数句柄  

应用于输入图像的等比例(各向同性)缩放范围，指定为以下形式之一：  
  
- 2元素数值向量： 第二个元素必须大于或等于第一个元素。缩放因子将从指定区间内的连续均匀分布中随机选取。  
- 函数句柄 : 该函数必须无输入参数，并返回一个数值标量作为缩放因子。可通过函数句柄实现从不连续区间或非均匀概率分布中选取缩放因子。 

默认情况下，变换 [tform](#P1) 不执行图像缩放操作。

**示例：** [0.5 4]

**XShear — 水平错切角度范围**  
[0 0]（默认）| 2 元素数值向量 | 函数句柄  

应用于输入图像的水平错切角度范围，单位为度，取值范围为(-90, 90)，指定为以下形式之一：  
  
- 2元素数值向量 : 第二元素须大于等于第一元素。水平错切角度将从指定区间内的连续均匀分布中随机选取。  
- 函数句柄 : 该函数必须无输入参数，并返回一个数值标量作为水平错切角度。可通过函数句柄实现从不连续区间或非均匀概率分布中选取水平错切角度。  

默认情况下，变换 [tform](#P1) 不会对图像执行水平方向错切操作。

**示例：** [0 45]  
  
**YShear — 垂直错切角度范围**  
[0 0]（默认）| 2元素数值向量 | 函数句柄  

应用于输入图像的垂直错切角度范围，单位为度，取值范围为(-90, 90)，指定为以下形式之一：  
  
- 2元素数值向量 : 第二元素须大于等于第一元素。垂直错切角度将从指定区间内的连续均匀分布中随机选取。

默认情况下，变换 [tform](#P1) 不会对图像执行垂直方向错切操作。

**示例：** [0 45]

**XTranslation — 水平平移范围**  
[0 0] (默认) | 2 元素数值向量 | 函数句柄  

应用于输入图像的水平平移范围，单位为像素，指定为以下形式之一 :  
  
- 2元素数值向量 : 第二元素须大于等于第一元素。水平平移距离将从指定区间内的连续均匀分布中随机选取。  
- 函数句柄 : 该函数必须无输入参数，并返回一个数值标量作为水平平移距离。可通过函数句柄实现从不连续区间或非均匀概率分布中选取水平平移距离。

**示例：** [-5 5]  
  
**YTranslation — 垂直平移范围**

应用于输入图像的垂直平移范围，单位为像素，指定为以下形式之一 :

**示例：** [-5 5]

### 输出参数
**<a id="P1"></a> tform — 仿射变换**  
`affinetform2d` 对象  

仿射变换，指定为 <a href="../affinetform2d/affinetform2d.html">affinetform2d</a>  </a> 对象。
