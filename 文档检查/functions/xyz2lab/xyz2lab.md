## xyz2lab
将 CIE 1931 XYZ 转换为 CIE 1976 L\*a\*b\*

## 简介
[`lab = xyz2lab(xyz)`](#function1)  
[`lab = xyz2lab(xyz, 'WhitePoint', whitePoint)`](#function2)

## 用法
<a id="function1"></a>
[lab](#P1) = xyz2lab[(xyz)](#Q1) 将 CIE 1931 XYZ 值（2° 观察者）转换为 CIE 1976 L\*a\*b\* 值。

<a id="function2"></a>
[lab](#P1) = xyz2lab([xyz](#Q1), 'WhitePoint', [whitePoint](#Q2)) 指定光源的参考白点。

## 参数说明
### 输入参数
**<a id="Q1"></a> XYZ — XYZ 颜色值**  
数值数组

要转换的 XYZ 颜色值，指定为数值数组，采用下列格式之一。

- c×3 颜色图。每一行指定一个 XYZ 颜色值。
- m×n×3 图像。
- m×n×3×p 图像堆叠。

**数据类型：** `single` | `double`

**<a id="Q2"></a> whitePoint — 参考白点**  
"d65" | "a" | "c" | "e" | "d50" | "d55"

参考白点，指定为 1×3 向量，或表中列出的某一种 CIE 标准光源。

| **取值**  | **白点（White Point）**                                                                                                  |
| ------- | -------------------------------------------------------------------------------------------------------------------- |
| `"a"`   | CIE 标准 A，[1.0985, 1.0000, 0.3558]。模拟典型的家用钨丝灯照明，相关色温为 2856 K。                                                |
| `"c"`   | CIE 标准 C，[0.9807, 1.0000, 1.1822]。模拟平均或北方天空日光，相关色温为 6774 K标准已被 CIE 弃用。                       |
| `"e"`   | 等能量辐射体，[1.000, 1.000, 1.000]。常用作理论参考白点。                                                     |
| `"d50"` | CIE 标准 D50，[0.9642, 1.0000, 0.8251]。模拟日出或日落时较暖的日光，相关色温为 5003 K。也称为地平线光。                      |
| `"d55"` | CIE 标准 D55，[0.9568, 1.0000, 0.9214]。模拟上午中段或下午中段日光，相关色温为 5500 K。                                             |
| `"d65"` | CIE 标准 D65，[0.9504, 1.0000, 1.0888]。模拟正午日光，相关色温为 6504 K。                                                    |

**数据类型：** `single` | `double` | `char`

### 输出参数
**<a id="P1"></a> lab — 转换后的 L\*a\*b\* 颜色值**  
数值数组

转换后的 L\*a\*b\* 颜色值，返回与输入大小和数据类型相同的数值数组。

| 属性 | 描述 |
| :----------------------- | :----------------------- |
| L\* | 图像的亮度。值在 [0, 100] 范围内，其中 0 指定黑色，100 指定白色。随着 L\* 的增加，颜色变得更亮。 |
| a\* | 图像中红色或绿色色调的量。大的正 a\* 值对应于红色或品红色。大的负 a\* 值对应于绿色。虽然 a\* 没有单一范围，但值通常在 [-100, 100] 或 [-128, 127) 范围内。 |
| b\* | 图像中黄色或蓝色色调的量。大的正 b\* 值对应于黄色。大的负 b\* 值对应于蓝色。虽然 b\* 没有单一范围，但值通常在 [-100, 100] 或 [-128, 127) 范围内。 |

**数据类型：** `single` | `double`
