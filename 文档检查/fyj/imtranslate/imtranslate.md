## imtranslate
平移图像

## 简介
[`B = imtranslate(A, translation)`](#function1)  
[`___ = imtranslate(___, method)`](#function2)  
[`___ = imtranslate(___, Name, Value)`](#function3)

## 用法 
<a id="function1"></a> [B](#P1) = imtranslate([A](#Q1), [translation](#Q2)) 根据 `translation` 中指定的二维或三维平移向量对图像 `A` 进行平移。如果 `A` 是三维图像且 `translation` 是二维向量，则 `imtranslate` 会将二维平移应用于 `A` 的每个平面。  
<a id="function2"></a> \_\_\_ = imtranslate(\_\_\_, [method](#Q3)) 使用 `method` 指定的插值方法对图像 `A` 进行平移。  
<a id="function3"></a> \_\_\_ = imtranslate(\_\_\_, [Name, Value](#Q4)) 对输入图像进行平移，并使用名称-值参量来控制平移的各个方面。

## 参数说明
### 输入参数
**<a id="Q1"></a> A — 需要平移的图像**  
数值数组 | 逻辑数组

需要平移的图像，指定为数值数组或逻辑数组。

**数据类型：**`single` | `double` | `int8` | `int16` | `int32` | `uint8` | `uint16` | `uint32` | `logical`

**<a id="Q2"></a> translation — 平移向量**  
2 元素数值向量 | 3 元素数值向量

平移向量，指定为 2 元素数值向量 (Tx Ty) 或 3 元素数值向量 (Tx Ty Tz)。值可以是小数。

**数据类型：**`single` | `double` | `int8` | `int16` | `int32` | `uint8` | `uint16` | `uint32` 

**<a id="Q3"></a> method — 插值方法**  
`"bilinear"`(默认) | `"nearest"` | `"bicubic"`

插值方法，指定为 `"bilinear"` (默认)、`"nearest"` 或 `"bicubic"`。

| **值**     | **描述**                                                                                                        |
| :--------- | :-------------------------------------------------------------------------------------------------------------- |
| `"nearest"`  | 最近邻点插值。赋给输出像素的值就是输入点所在像素的值。不考虑其他像素。                                          |
| `"bilinear"` | 双线性插值。输出像素值是最近的 2 × 2 邻域中像素的加权平均值。                                                     |
| `"bicubic"`  | 双三次插值。输出像素值是最近的 4 × 4 邻域中像素的加权平均值。（注意：双三次插值可能生成在原始范围之外的像素值。） |

**数据类型：**`char` | `string` |

### <a id="Q4"></a> 名称-值参数
将可选的参量对组指定为 Name1, Value1, ... , NameN, ValueN，其中 `Name` 是参量名称，`Value` 是对应的值。名称-值参量必须出现在其他参量后，但对各个参量对组的顺序没有要求。

示例：mriVolumeTranslated = imtranslate(mriVolume, [40 30 0], "OutputView", "full");  
**OutputView — 输出世界范围**  
`"same"`(默认) | `"full"`

输出世界坐标系范围，指定为以下值之一。

| **值** | **描述**                                                   |
| :----- | :--------------------------------------------------------- |
| `"same"` | 输出世界系范围与输入图像相同（默认）。                       |
| `"full"` | 输出世界坐标系范围是包含输入图像和平移后输出图像的边界矩形。 |

**数据类型：**`char` | `string`

**FillValues — 填充值**  
数值标量 | 数值数组

用于输入图像以外的输出像素的填充值，指定为表中的值之一。如果输出像素对应的位置经逆变换后完全处于输入边界之外，`Imtranslate` 对输出像素使用填充值。  
数值和逻辑图像的默认填充值为 0。

| **图像类型**                 | **变换维度** | **填充值的格式**                                                                                                                        |
| :--------------------------- | :----------- | :-------------------------------------------------------------------------------------------------------------------------------------- |
| 二维灰度或逻辑图像           | 二维         | - 数值标量                                                                                                                              |
| 二维彩色图像或二维多光谱图像 | 二维         | - 数值标量 <br> - 包含 c 个元素的数值向量，为 c 个通道中的每个通道指定一个填充值。对于彩色图像，通道数 c 为 3                           |
| 包含 p 个二维图像的序列      | 二维         | - 数值标量 <br> - c×p 数值矩阵。对于灰度图像，通道数 c 为 1，对于彩色图像为 3                                                           |
| N 维图像                     | 二维         | - 数值标量 <br> - 数值数组，其大小与输入图像 [A](#Q1) 的维度 3 至 N 相匹配。例如，如果 `A` 是 200×200×10×3，则 FillValues 可以是 10×3 数组。 |
| 三维灰度或逻辑图像           | 三维         | - 数值标量                                                                                                                              

**示例:** 255 表示用白色像素填充 `uint8` 图像

**示例:** 1 表示用白色像素填充 `double` 图像

**示例:** [0 1 0] 表示用绿色像素填充 `double` 彩色图像

**示例:** 对于一个包含两个 `double` 彩色图像的序列，(0 1 0; 0 1 1) 表示用绿色像素填充第一个图像，用青色像素填充第二个图像。

**数据类型：**`single` | `double` | `int8` | `int16` | `int32` | `int64` | `uint8` | `uint16` | `uint32` | `uint64` | `char` | `string` 

### 输出参数
**<a id="P1"></a> B — 平移后的图像**  
数值数组 | 逻辑数组

平移后的图像，指定为与输入图像 [A](#Q1) 数据类型相同的数值数组或逻辑数组。
