## imdrawlines
在图像上画线

## 简介
[`B = imdrawlines(I, p1, p2)`](#function1)  
[`B = imdrawlines(I, p1, p2, color)`](#function2)

## 用法 
<a id="function1"></a> [B](#P1) = imdrawlines([I](#Q1), [p1](#Q2), [p2](#Q3)) 在图像 `I` 上绘制以 `p1`、`p2` 为端点的直线。  
<a id="function2"></a> [B](#P1) = imdrawlines([I](#Q1), [p1](#Q2), [p2](#Q3), [color](#Q4)) 使用 `color` 来指定线的颜色。

## 参数说明
### 输入参数
**<a id="Q1"></a> I — 需要画线的图像** 


需要画线的图像，指定为灰度、真彩色或二值图像。

**<a id="Q2"></a> p1 — 直线的第一个端点**  


直线的第一个端点，指定为正数或由正数组成的列向量，长度与 p2 相同。

**<a id="Q3"></a> p2 — 直线的第二个端点**  


直线的第二个端点，指定为正数或由正数组成的列向量，长度与 p1 相同。

**<a id="Q4"></a> color — 线的颜色**  


线的颜色，现仅支持指定为 RGB 三元组。

RGB 三元组是包含三个元素的行向量，其元素分别指定颜色中红、绿、蓝分量的强度。强度值必须位于 [0, 255] 范围内，例如 [100 200 30]。

### 输出参数

**<a id="P1"></a> B — 画上线后的图像**  


画上线后的图像，返回为 RGB 图像。
