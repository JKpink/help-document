## imfuse
将 2 幅图像合成

## 简介
[`C = imfuse(A, B)`](#function1)  
[`C = imfuse(___, method)`](#function3)  
[`C = imfuse(___, Name, Value)`](#function4)  
<!-- [`[C RC] = imfuse(A, RA, B, RB)`](#function2)   -->

## 用法 
<a id="function1"></a> [C](#P1) = imfuse([A](#Q1), [B](#Q2)) 融合两幅输入图像（`A` 和 `B`）生成输出矩阵 `C`。在融合流程中，若检测到尺寸不匹配，函数将对较小图像进行边界零填充以实现维度对齐。最终输出的数值矩阵 `C` 即为两幅图像在统一维度空间下的融合数据表示。  
<a id="function3"></a> [C](#P1) = imfuse(\_\_\_, [method](#Q5)) 使用 `method` 指定的算法来创建合成图像。  
<a id="function4"></a> [C](#P1) = imfuse(\_\_\_, [Name, Value](#Q6)) 支持任何上述语法，且可使用一个或多个名称-值参数对指定其他附加项。    
<!-- <a id="function2"></a> [[C](#P1) [RC](#P2)] = imfuse([A](#Q1), [RA](#Q3), [B](#Q2), [RB](#Q4)) 使用 `RA` 和 `RB` 提供的空间参考信息，将图像A和B合成一幅图像。输出 `RC` 定义了融合图像 `C` 的空间参考信息。   -->

## 参数说明
### 输入参数
**<a id="Q1"></a> A — 要进行合成的第一幅图像**  
灰度图像 | 真彩色图像 | 二值图像

要进行合成的第一幅图像，可指定为灰度图像，真彩色图像或二值图像。

**数据类型：**`single` | `double` | `int8` | `int16` | `int32` | `int64` | `uint8` | `uint16` | `uint32` | `uint64` | `logical`

**<a id="Q2"></a> B — 要进行合成的第二幅图像**  
灰度图像 | 真彩色图像 | 二值图像

要进行合成的第二幅图像，可指定为灰度图像，真彩色图像或二值图像。

**数据类型：**`single` | `double` | `int8` | `int16` | `int32` | `int64` | `uint8` | `uint16` | `uint32` | `uint64` | `logical`

<!-- **<a id="Q3"></a> RA — 与输入图像A关联的空间参考信息。**  
imref2d 对象

与输入图像A关联的空间参考信息，指定为 `imref2d` 对象。

**<a id="Q4"></a> RB — 与输入图像B关联的空间参考信息。**  
imref2d 对象

与输入图像B关联的空间参考信息，指定为 `imref2d` 对象。 -->

**<a id="Q5"></a> method — 用于合成图像的算法**  
"falsecolor"（默认）| "blend" | "diff" | "montage"

用于组合图像的算法，指定为以下值之一。

| **方法**       | **描述**                                                                                                                                            |
| :------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------- |
| "falsecolor"   | 用不同色带叠加 `A` 和 `B`。灰色区域表示图像具有相同强度的位置。彩色区域表示图像强度值不同的位置。 |
| "blend"        | 使用 Alpha 混合叠加 `A` 和 `B`。显示的强度是两个图像的均值。                                                                                                                 |
| "checkerboard" | 使用来自 `A` 和 `B` 的交替矩形区域显示图像。                                                                                                          |
| "diff"         | 显示 `A` 和 `B` 的差异图像。该函数在计算差异图像之前会将 RGB 图像转换为灰度图像。                                                                                                                    |
| "montage"      | 将 `A` 和 `B` 在同一图窗中并排放置。                                                                                                                  |

### <a id="Q6"></a> 名称-值参数
将可选的参量对组指定为Name1, Value1, ... , NameN, ValueN，其中 Name 是参量名称，Value 是对应的值。名称-值参量必须出现在其他参量后，但对各个参量对组的顺序没有要求。  
示例：C = imfuse(A, B, "Scaling", "joint") 将 `A` 和 `B` 的强度值作为一个数据集进行统一缩放。

**Scaling — 强度缩放选项**  
"independent"（默认）| "joint" | "none"

强度缩放选项，指定为以下列值之一。

| **选项**      | **描述**                                                                                                                                             |
| :------------ | :--------------------------------------------------------------------------------------------------------------------------------------------------- |
| "independent" | 彼此独立地缩放 `A` 和 `B` 的强度值。                                                                                                            |
| "joint"       | 将两幅图像视为同一数据整体进行强度归一化。<br>该模式特别适用于单模态图像的配准可视化：如果一个图像包含的填充值在另一个图像的动态范围之外，联合缩放可有效抑制填充值对全局对比度的干扰，确保有效区域获得最佳可视化对比。 |
| "none"        | 不进行缩放。                                                                                                                                     |

**ColorChannels —每个输入图像的输出颜色通道**  
"green-magenta"（默认）| [R G B] | "red-cyan"  

每个输入图像的输出颜色通道，指定为以下选项之一。

| **选项**        | **描述**                                                                                                                                                   |
| :-------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [R G B]         | 三元向量，用于指定合成图像中红、绿、蓝三个输出通道分别对应哪个输入图像。R、G、B的取值为 1（应用于第一个输入图像）、2（应用于第二个输入图像）和 0（不应用于任何输入图像）。 |
| "red-cyan"      | 向量 [1 2 2] 的快捷方式，适用于红色/青色立体浮雕。                                                                                                         |
| "green-magenta" | 向量 [2 1 2] 的快捷方式，这是高对比度选项，非常适合多种颜色色盲患者。                                                                                      |

### 输出参数
**<a id="P1"></a> C — 融合图像**  
灰度图像 | 真彩色图像 | 二值图

融合图像是输入图像的合成结果，返回为灰度图像、真彩色图像或二值图像。

**数据类型：**`unit8`
